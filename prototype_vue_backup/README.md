# SmartCON Lite 프로토타입

프로토타입 버전입니다. 실제 데이터 없이 화면 위주로 구현되었습니다.

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 주요 기능

- 사용자 선택 로그인 (프로토타입 모드)
- 역할 선택 (다중 역할 지원)
- 역할별 대시보드
- 출역 체크 (시뮬레이션)
- 안면 등록 (5단계 프로세스)
- 작업일보 작성/조회
- 마이페이지 (출역 현황, 이의 제기)

## 프로토타입 특징

- 실제 API 연동 없음
- 목업 데이터 사용
- 모든 기능을 클릭/링크로 동작
- PC/모바일 반응형 디자인
