# SmartCON Lite MVP 완성 작업 계획서

## 작업 개요

SmartCON Lite 프로젝트를 2025년 2월 15일까지 MVP 수준으로 완성하기 위한 상세 작업 계획입니다. 현재 44% 완성도에서 프로덕션 배포 가능한 수준까지 개발을 진행합니다.

## 팀 구성 및 역할 분담

### 웹 개발자 (1명)
- **주요 담당**: 백엔드 API 개발 (60%) + 웹 프론트엔드 (40%)
- **핵심 업무**: Spring Boot API, JWT 인증, 데이터베이스 마이그레이션, 웹 UI 연동

### 모바일 앱 개발자 (1명)
- **주요 담당**: 모바일 앱 개발 (70%) + 웹 지원 (30%)
- **핵심 업무**: Capacitor 앱, 네이티브 기능, 웹 UI 완성도 향상

## 바이브코딩(AI 도구) 활용 전략

### 코드 생성 및 자동화 (40-60% 생산성 향상 예상)
- **백엔드**: Spring Boot 보일러플레이트 코드, JPA 엔티티, REST API 컨트롤러
- **프론트엔드**: React 컴포넌트, TypeScript 인터페이스, API 클라이언트 코드
- **테스트**: 단위 테스트 및 통합 테스트 코드 자동 생성
- **문서화**: API 문서, 사용자 가이드 초안 생성

### AI 도구 사용 가이드라인
1. **코드 리뷰**: AI 생성 코드는 반드시 수동 검토 후 적용
2. **보안 검증**: 보안 관련 코드는 추가 검증 필수
3. **테스트 검증**: AI 생성 테스트 코드의 실제 동작 확인
4. **문서 검토**: AI 생성 문서의 정확성 및 완성도 검증
