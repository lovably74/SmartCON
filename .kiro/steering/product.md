# SmartCON Lite Product Overview

SmartCON Lite is a SaaS-based construction workforce management platform that automates attendance tracking using FaceNet-based facial recognition technology.

## Core Value Proposition
**"현장의 신뢰, 기술로 완성하다" (Building Trust Through Technology)**

- **Transparency**: Objective attendance management through facial recognition
- **Convenience**: Mobile-first intuitive user experience  
- **Flexibility**: Adaptable system for various construction site environments

## Key Features
- **Facial Recognition Attendance**: FaceNet-based automated check-in/check-out
- **Multi-tenant SaaS**: Company-level data isolation with subscription billing
- **Role-based Access**: 5-tier user roles (Super Admin, HQ Admin, Site Manager, Team Leader, Worker)
- **Electronic Contracts**: Digital contract signing with PDF generation
- **Real-time Dashboards**: Role-specific dashboards with live attendance monitoring
- **Work Report Management**: Automated daily reports with weather API integration
- **Mobile-first Design**: Hybrid app using React + Capacitor

## Target Users
- Construction companies needing workforce management
- Site managers requiring real-time attendance tracking
- Team leaders managing worker assignments
- Workers needing simple attendance and contract management

## Business Model
- Subscription-based SaaS with multi-tenant architecture
- Pay-per-site or pay-per-worker pricing models
- Automated billing with tax invoice generation
